Для того щоб сформувати необхідні параметри для мапи необхідно примінити фільтри чи вибрати необхідний обʼєкт й далі копіючи код вставляти його в iframe. Приклади та необхідний код наведені нижче.

PS: не весь функціонал може генеритись в параметри url, тому якщо його не має, але цей функціонал потрібен, тоді треба йодо додати з фронтової частини мапи дадотково. Для початку додав головні фічі.

Звичайна карта

Code:
<iframe id="inlineFrameExample" title="Inline Frame Example" src="https://staging.map.dream.gov.ua" > </iframe>

Карта по харчоблокам

Code:
<iframe id="inlineFrameExample" title="Inline Frame Example" src="https://staging.map.dream.gov.ua/?activeFilter=foodBlock&filter_foodBlock=basic&filter_foodBlock=sustaining&filter_foodBlock=diningArea" > </iframe>

Тип балансоутримувача: Адміністрація

Code:
<iframe id="inlineFrameExample" title="Inline Frame Example" src="https://staging.map.dream.gov.ua/?activeFilter=balanceHolderType&filter_balanceHolderType=administration" > </iframe>

Виділений обʼєкт за допомогою code id

Code:
<iframe id="inlineFrameExample" title="Inline Frame Example" src="https://staging.map.dream.gov.ua/?object_id=DREAM-UA-051023-1AA151CE" > </iframe>

Виділений фільтр з РПЗМ та обʼєкт за допомогою code id

Code:
<iframe id="inlineFrameExample" title="Inline Frame Example" src="https://staging.map.dream.gov.ua/?activeFilter=registrationRpzm&filter_registrationRpzm=rpzm&object_id=DREAM-UA-270923-6B5919CF" > </iframe>
